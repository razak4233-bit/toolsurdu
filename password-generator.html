❓ Aksar Pooche Jane Wale Sawal — FAQ

Q: Strong password kya hota hai aur kaise pehchanen?
A: Strong password woh hota hai jisme kam az kam 12 characters hon, uppercase aur lowercase dono hon, numbers hon aur special symbols (!@#$%) bhi hon. Hamare tool mein strength indicator automatically batata hai password kitna strong hai — Weak, Fair, Good, ya Strong.
Q: Kya yeh passwords safe hain? Koi save karta hai kya?
A: Bilkul safe. Password sirf aapke browser mein generate hota hai — koi server, koi database mein save nahi hota. Hamare paas koi record nahi rehta. Aapke ilawa koi nahi dekh sakta.
Q: Kitna lamba password hona chahiye?
A: Minimum 12 characters. Email, bank, ya social media ke liye 16-20 characters ideal hain. Gaming ya kam important accounts ke liye 12 bhi theek hai. Hamare tool mein 64 characters tak generate kar sakte hain.
Q: Password yaad nahi rahega — kya karun?
A: Password manager use karo. Bitwarden bilkul free hai aur open source bhi — sab platforms pe kaam karta hai. Google Password Manager bhi free hai aur Chrome ke saath integrated hai. Sirf ek master password yaad rakhna hoga.
Q: Kya main har account ke liye alag password use karun?
A: Haan — yeh sabse zaroori rule hai. Agar aap ek hi password sab jagah use karte hain aur ek site hack ho jaye, toh aapke sab accounts khatre mein aa jaate hain. Hamare tool se alag alag passwords banao aur password manager mein save karo.
Q: Symbols kyun zaroori hain password mein?
A: Symbols (!@#$%^&*) password ko exponentially zyada strong banate hain. Sirf numbers aur letters use karne se hackers "brute force" attack se password crack kar sakte hain. Symbols ke saath yeh practically impossible ho jata hai.
Q: Two-factor authentication (2FA) kya hoti hai?
A: 2FA ek extra security layer hai — password ke saath ek OTP bhi chahiye login ke liye. Strong password ke saath 2FA enable karo toh account practically unhackable ho jata hai. Gmail, Facebook, aur banking apps sab mein 2FA available hai.
